# Urdu Learning Platform - DevOps Deployment

## FA23-BCS-035 | Arfa Riaz

A full-stack Node.js + MongoDB application containerized with Docker and deployed on Kubernetes via Azure.

## Tech Stack
- Node.js + Express
- MongoDB Atlas
- Docker
- Kubernetes (AKS)
- GitHub for version control

## Run Locally
```bash
npm install
node server.js
```

## Run with Docker
```bash
docker run -p 3000:3000 fa23-bcs-035
```
